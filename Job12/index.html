Информация о плагине adjust

С помощью плагина adjust Вы можете изменять цвет и размер шрифта выбранных элементов.

Для того, чтобы подключить плагин adjust необходимо добавить следующий код в секцию head: .

С помощью опции size плагина Вы можете задать размер шрифта, а с помощью опции color цвет. По умолчанию опция color имеет значении green, а опция size 1.4em.
